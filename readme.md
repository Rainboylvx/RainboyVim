# RainboyVim - lua neovim

在vim9 与 neovim 中,我最后还是选择了neovim

## 安装

确保`~/.config/`路径下没有`nvim`文件夹

```bash
mv ~/.config/nvim ~/.config/nvim_bak
git clone -b lua-neovim https://github.com/Rainboylvx/RainboyVim ~/.config/nvim
```
