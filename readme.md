# RainboyVim

## 1.下载 或 手动安装 `vim 8.0


## 2. clone 本配置


```bash
# clone 到~
git clone https://github.com/Rainboylvx/RainboyVim.git ~
cd ~/RainboyVim
# install font: DejaVu Sans Mono for Powerline Nerd Font Plus Font Awesome Plus Octicons Plus Pomicons Windows Compatible.ttf
cp ./extra/.vimrc ~
```

## 3. 安装Plugin

```bash
# 进入vim
vim

# 安装Plugin
:PluginInstall
```
